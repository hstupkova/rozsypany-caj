# Rozsypaný čaj

Závěrečný projekt v kurzu Staň se kodérkou. Obsahem kurzu bylo naučit se:

* HTML
  * sémantické značky
* CSS
  * box model
  * margin a padding
  * display
  * pozicování
  * flexbox
  * grid
  * pseudoprvky
  * pokročilé CSS selektory
  * CSS specificita
  * transformace a animace
* Responzivní webdesign
  * mobile first
  * media query
* Formuláře
  * jednoduchá validace
  * stylování
* Grafické formáty
  * ratrová a vektorová grafika
  * SVG
* BEM
* Sass
* generátor statických webů 11ty
* domény, webhosting, publikování webu
* Netlify
* základy UX
  * principy
  * wireframes
  * uživatelské testování
* Git
